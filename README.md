## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Hanish0925/AT_Frontend.git
    ```

2. Navigate to the project directory:
    ```bash
    cd AT_Frontend
    ```

3. Open `IPL_Stats.html` in your browser to view the webpage.
